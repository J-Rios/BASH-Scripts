
MyTools Scripts list:
--------------------------------------
mytools_update - Update actual installed mytools scripts to lastest version.
mytools_help - Shows mytools scripts list and descriptions.
copy - Set a file/directory into clipboard (Ctrl+C).
paste - Copy the file/directory from clipboard into a specific path (Ctrl+V).
epoch - Shows seconds since 1970 or since provided year.
fencrypt - Encrypt a file with AES256 and provided passphrase.
fdecrypt - Decrypt and show an AES256 encoded file.
ntp_get - Get and shows date from NTP (check NTP Server connection).
numtoint - Convert a binary or hexadecimal value into decimal.
hextoint - Convert hexadecimal to decimal value.
bintoint - Convert binary to decimal value.
numtohex - Convert a decimal or binary value into hexadecimal.
inttohex - Convert decimal to hexadecimal value.
bintohex - Convert binary to hexadecimal value.
numtobin - Convert a decimal or hexadecimal value into binary.
inttobin - Convert decimal to binary value.
hextobin - Convert hexadecimal to binary value.
hextob64 - Encode hexadecimal data into Base-64.
b64tohex - Decode from Base-64 to hexadecimal.
imgflash - Write an .img disk file content into a device (SD/USB/HDD/etc).
imgread - Create an .img disk file from a device (SD/USB/HDD/etc).
gitdiff - Git diff between two files without needs GIT repositories.
gitlog - Pretty show git log.
check_same_lines - Check for coincidence of text lines between two files.
net_check_devices - Make a network scan to check connected devices/systems.
processcheck - Check for running process info.
mqtt_log - MQTT ease logger tool.
timescp - Automate download of a file from a server each X minutes.
dns_block_trackers - Install/Update hosts-blocklists files and setup dnsmasq to blacklist that domains.
show_lorawan_activation_keys - Shows LoRaWAN activation keys requeriments (which keys needs to be known for each network element).
show_lorawan_regions_rx2 - Shows a list of default LoRaWAN Rx2 reception Window regional parameters (Data Rate & Frequency).
show_xdot_at - Shows a list of Multitech MTXDOT modems AT commands with description.
